Python tools

This page is intended to be a minimal overview of the Anaconda distribution and jupyter-lab interface. For more detailed guides have a look at Anaconda's official documentation.

Anaconda Navigator

The Anaconda distribution supports a large number of python tools and to simplify their use, the distribution includes a small application called the Anaconda Navigator. For our purposes, there are two important items:

  • The Environments tab, on the left, that opens graphical interface to the conda package manager.
  • The jupyter-lab card near the top right, that starts the jupyter-lab kernel back-end and opens the front-end in a browser window.
