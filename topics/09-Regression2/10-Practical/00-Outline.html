Linear Regression Using Python

This week, we consider regression having more advanced aspects:

  • use of regularisation when some predictors are correlated and hence too similar to each other
  • use of cross-validation for
  • model building: deciding the polynomial degree to predict a target
  • hyperparameter tuning: trying to find the best value of the regularisation parameter lambda
  • categorical predictors (that need to be transformed to numeric predictors)

The first example relates to fitting a polynomial to data, with or without regularisation, using cross-validation to derive a test score that can be compared for different polynomial degrees.

The next example relates to the analysis of the Credit Balance dataset. I provide a detailed notebook with examples and discussion of how to perform linear regression with categorical-valued predictors, how to use regularisation to build models (when the model coefficients indicate that the solution is unstable) and how to use cross-validation to find the optimum value of the regularisation tuning parameter lambda.
